# word_frequency_by_year
# 연도별 단어 빈도 추출 (Word Frequency by Year)

연도별 문맥 파일(`.context`)로부터 각 단어의 연도별 출현 빈도를 추출하여 정렬된 형태로 출력하는 Python 스크립트입니다.

---

## 파일 구성

```
.
├── word_frequency_by_year.py   # 연도별 단어 빈도 추출 스크립트 (메인)
├── word_frequency.py           # 단일 파일 단어 빈도 추출 스크립트 (참고용)
├── 2000_tag.context            # 연도별 문맥 파일 (2000 ~ 2019)
├── 2001_tag.context
│   ...
├── 2019_tag.context
└── result.txt                  # 실행 결과 출력 파일
```

---

## 요구 사항

- Python 3.x
- 표준 라이브러리만 사용 (`sys`, `re`, `collections`) — 별도 패키지 설치 불필요

---

## 실행 방법

```bash
# 실행 권한 부여 (최초 1회)
chmod +x word_frequency_by_year.py

# 실행 (결과를 파일로 저장)
./word_frequency_by_year.py *.context > result.txt

# 또는 python3로 직접 실행
python3 word_frequency_by_year.py *.context > result.txt
```

> 처리 진행 상황은 `stderr`로 출력되므로, `> result.txt` 리다이렉션과 무관하게 터미널에 표시됩니다.

---

## 출력 형식

탭(`\t`)으로 구분된 **단어**와 **연도별 빈도 리스트**를 한 줄씩 출력합니다.  
빈도 리스트의 각 원소는 입력 파일의 연도 순서에 대응합니다.

```
단어<TAB>[연도1 빈도, 연도2 빈도, ..., 연도N 빈도]
```

### 출력 예시 (2000 ~ 2019년, 20개 연도)

```
%포인트	[2, 3, 0, 1, 4, 4, 1, 2, 3, 4, 5, 5, 0, 2, 2, 3, 4, 1, 7, 4]
컴퓨터	[70, 46, 29, 35, 32, 36, 26, 22, 19, 24, 18, 20, 15, 20, 16, 14, 20, 5, 9, 11]
```

- 리스트 인덱스 0 → 가장 이른 연도 (예: 2000년)
- 리스트 인덱스 19 → 가장 늦은 연도 (예: 2019년)
- 해당 연도에 등장하지 않은 단어의 빈도는 `0`으로 표시
- 출력은 **단어순(알파벳/가나다 순)으로 정렬**

---

## 동작 방식

1. **연도 추출**: 파일명에서 정규식으로 4자리 연도를 파싱합니다.  
   예) `2000_tag.context` → `2000`

2. **연도 정렬**: 입력 파일의 전달 순서와 무관하게 연도 오름차순으로 정렬합니다.

3. **단어 빈도 계산**: 각 파일을 줄 단위로 읽어 공백(whitespace)으로 분리한 뒤, 단어별 출현 횟수를 집계합니다.

4. **전체 단어 집합 구성**: 모든 연도의 파일에 등장한 단어를 합집합으로 수집합니다.

5. **정렬 후 출력**: 단어를 사전순으로 정렬하고, 각 단어에 대해 연도별 빈도 리스트를 탭 구분 형식으로 출력합니다.

---

## 입력 파일 형식

각 `.context` 파일은 한 줄에 하나의 문맥(context)이 담긴 텍스트 파일입니다.  
줄 내의 토큰은 공백으로 구분되어 있습니다.

```
김현욱 金顯煜 사무총장 최근 공동정권 합의정신 수도권 양당 지분 ５대 기준 논의 강조
결국 학교 자퇴
부모 문제 대처
...
```

| 항목 | `word_frequency.py` | `word_frequency_by_year.py` |
|------|--------------------|-----------------------------|
| 입력 | 단일 또는 복수 파일 | 복수 파일 (연도별) |
